# Microsoft-Excel-Data-Analysis-and-Dashboard-Reporting
Interactive Excel dashboard for customer order analysis and reporting. Includes KPIs, slicers, and dynamic visuals built using PivotTables, charts, and formulas. Demonstrates professional data analysis and dashboard-building skills.

## Dashboard Overview
<img width="1323" height="707" alt="image" src="https://github.com/user-attachments/assets/1290ae7e-ee82-45cb-8574-ef189dc4d1ce" />


This dashboard showcases:
- **Customer Details** (Name, Contact, Address)
- **KPIs:** Order Count, Average Freight, and Last Order Date
- **Interactive Slicers** for Month and Year selection
- **Dynamic Charts** to visualise yearly order totals
- **Order History Table** updated automatically with filters

Skills Applied
- Analyzing Data & Validation Functions (`PROPER`, `UPPER`, `CHOOSE`, `TEXT`, `IF()` )
- Creating a Drop Down Menu with Data Validation.
- Advanced Excel Formulas (`LOOKUP`, `INDEX-MATCH`, `IFERROR`)
- Record MACRO for Advanced Filter
- VBA (Visual Basic for Application) and Modification of VBA code
- PivotTables & PivotCharts  
- Data Filtering and Conditional Formatting  
- Dynamic Dashboard Design using Slicers and Buttons  
- Business Insights Reporting
